Quick Start

  1. Enter your documentation structure in JSON format
  2. Click "Create & Download Zip" button
  3. Extract the ZIP into your Nextra project's pages directory

JSON Structure Format

The structure follows these rules:

  • Each object key represents a folder
  • Keys starting with "--" become section titles
  • Arrays contain the MDX file names

Example Structure

{
  "Course Name": {
    "Chapter 1": {
      "--Introduction": [
        "What is the Course?",
        "Prerequisites"
      ],
      "--Getting Started": [
        "Installation Guide",
        "First Steps"
      ]
    }
  }
}

Generated Files

For each folder with files, you'll get:

  • A folder with the specified name
  • A _meta.js file for navigation
  • MDX files for each item in the arrays

_meta.js Example

const meta = {
  "Introduction": {
    title: "Introduction",
    type: "separator"
  },
  "what-is-the-course": "What is the Course?",
  "prerequisites": "Prerequisites"
};
